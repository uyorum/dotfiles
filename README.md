# dotfiles

## Support
* Emacs
* Zsh
* tmux
* Vim
* Git
* Ruby(Gem)
* GNU Global

## Specific to OS
### macOS
* Karabinier

### Windows
* AutoHotkey

### Linux
* Xmodmap

## Requirements
* Zsh
* Unzip

## Installation

```bash
$ curl https://raw.githubusercontent.com/uyorum/dotfiles/master/bin/setup.sh | bash
```
